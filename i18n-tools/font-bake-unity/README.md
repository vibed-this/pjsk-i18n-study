# font-bake-unity

从 Hub 可打开的 `unity-example/PJSK-Bake-Font` **整份复制**而来，烘焙逻辑一致。

Hub 添加：

```
E:\GithubRepos\pjsk-i18n-mod\i18n-tools\font-bake-unity
```

## 烘焙

```powershell
cd E:\GithubRepos\pjsk-i18n-mod\i18n-tools\font-bake-unity
.\bake.ps1
```

或 **PJSK → Font Bake (local)**

产物：`i18n/font/source-han-fallback.bundle`
